> _"I think about Roblox games like live-service platforms — not just places with scripts."_

I got curious about code around 10, started actually building toy things by 14, and in 2017 I cracked open Roblox Studio to make the games I wanted to play. A decade and a half in, the spark hasn't changed — the stack and standards just got sharper. I move between **game systems, full-stack product, AI tooling, and writing** — usually with a 15-doc GDD in front of me before I touch a single script.
